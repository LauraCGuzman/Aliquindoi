1. DESCRIPCIÓN GENERAL
═══════════════════════════════════════════════════════════════════════════════

Este programa procesa datos espectrales de muestras medidas con:
  - Espectrofotómetro UV-Vis (longitudes de onda: 280-2500 nm)
  - FTIR (infrarrojo)

Tipos de medidas soportadas:
  - Reflectancia
  - Absortancia
  - Transmitancia CSP
  - Transmitancia PV

═══════════════════════════════════════════════════════════════════════════════
2. ARCHIVOS DE ENTRADA (INPUTS)
═══════════════════════════════════════════════════════════════════════════════

2.1 ARCHIVOS DE MUESTRAS - ESPECTROFOTÓMETRO (.asc)
────────────────────────────────────────────────────
Deben nombrarse siguiendo este patrón:
  
  sample_<nombre_muestra>-<identificador>.Sample.asc
  
Ejemplos:
  ✓ sample_ABC123-001.Sample.asc
  ✓ sample_Muestra_Test-20260126.Sample.asc
  ✓ sample_Material-A-xyz.Sample.asc

IMPORTANTE:
  - El prefijo "sample_" es OBLIGATORIO
  - Debe contener al menos un guión "-" antes de la extensión
  - La parte entre "sample_" y el último "-" identifica la muestra
  - La extensión debe ser .Sample.asc


2.2 ARCHIVOS DE REFERENCIA - ESPECTROFOTÓMETRO (.asc)
──────────────────────────────────────────────────────
Para Reflectancia y Absortancia, se requieren:

  a) Línea base (baseline):
     base_<identificador>.BaseLine.asc
     Ejemplo: base_20260126.BaseLine.asc

  b) Línea cero (zeroline):
     zero_<identificador>.ZeroLine.asc
     Ejemplo: zero_20260126.ZeroLine.asc

  c) Ventana (opcional, si se mide con ventana):
     ventana_<identificador>.Sample.asc
     Ejemplo: ventana_cuarzo_20260126.Sample.asc

NOTA: Para Transmitancia CSP/PV no se requieren archivos base/zero.


2.3 ARCHIVOS FTIR (.xlsx o .xls)
─────────────────────────────────
a) Archivo de datos de muestras:
   Debe contener "data" en el nombre del archivo.
   Ejemplo: 20260126data.xlsx
   
   Dentro del archivo, las pestañas deben nombrarse:
   sample_<nombre_muestra>-<identificador>
   Ejemplo: sample_ABC123-001

b) Archivo de referencias:
   Debe contener la fecha y "ref" en el nombre.
   Ejemplo: 20260126ref.xlsx
   
   Dentro del archivo, se requieren pestañas con los siguientes prefijos:
   - zero_       (línea cero)
   - base_       (línea base)
   - ventana_    (si se usa ventana protectora)


2.4 ARCHIVO DE REFERENCIAS GENERALES (references.xlsx)
───────────────────────────────────────────────────────
Ubicación: user_templates/references.xlsx

Este archivo contiene las referencias espectrales para diferentes tipos de
materiales y configuraciones. Debe tener las siguientes pestañas:

  - absorbedores_abs      (referencias para medidas de absortancia)
  - absorbedores_refl     (referencias para medidas de reflectancia FTIR)
  - reflectores           (referencias para reflectores UV)
  - ref_trans_csp         (referencias para transmitancia CSP)
  - T_ventana_uv          (transmitancia de ventana UV)
  - T_ventana_ir          (transmitancia de ventana IR)
  - testsite              (lista de tipos de test disponibles)
  - fabricantes           (lista de fabricantes)
  - proyectos             (lista de proyectos)

Durante la ejecución, el programa solicitará seleccionar las columnas
apropiadas de estas pestañas.

3. ARCHIVOS DE SALIDA (OUTPUTS)
═══════════════════════════════════════════════════════════════════════════════

3.1 ARCHIVO EXCEL DE RESULTADOS
────────────────────────────────
El programa genera un archivo Excel (.xlsx) que contiene:

Para REFLECTANCIA:
  - Ubicación de plantilla: user_templates/plantillas/refl_plantilla.xls
  - Una pestaña por cada muestra procesada
  - Datos de reflectancia en función de la longitud de onda
  - Metadatos: nombre, fabricante, proyecto, fecha, test, etc.
  - Hasta 9 medidas por muestra

Para ABSORTANCIA:
  - Ubicación de plantilla: user_templates/plantillas/abs_plantilla.xlsx
  - Datos de absortancia y reflectancia en función de longitud de onda
  - Cálculos de:
    * SWR_UV (Solar Weighted Reflectance UV)
    * SWA_UV (Solar Weighted Absorptance UV)
    * Desviación estándar de SWR
    * Emitancia térmica
    * Temperatura
  - Tablas separadas para rangos UV e IR

Para TRANSMITANCIA CSP:
  - Ubicación de plantilla: user_templates/plantillas/trans_csp_plantilla.xls
  - Datos de transmitancia
  - Metadatos de la muestra

Para TRANSMITANCIA PV:
  - Ubicación de plantilla: user_templates/plantillas/trans_pv_plantilla.xls
  - Datos de transmitancia
  - Metadatos de la muestra


3.2 ARCHIVO DE LOG DE ERRORES
──────────────────────────────
En caso de error durante la ejecución, se generará un archivo .txt con el
mismo nombre que el archivo de salida Excel. Este archivo contiene:
  - Marca de tiempo del error
  - Traza completa del error (traceback)
  - Información para diagnóstico

Ejemplo: Si el archivo de salida es "resultados_20260126.xlsx", el log de
errores será "resultados_20260126.txt"

4. CONVENCIONES DE NOMENCLATURA
═══════════════════════════════════════════════════════════════════════════════

4.1 REGLAS GENERALES
────────────────────
  - Los nombres de muestras NO deben contener caracteres especiales
  - Se recomienda usar letras, números, y los separadores: guión (-) o 
    guión bajo (_)
  - El programa procesa los nombres eliminando espacios y algunos separadores
    para encontrar coincidencias


4.2 CORRESPONDENCIA ENTRE ARCHIVOS
───────────────────────────────────
El programa identifica automáticamente las muestras y busca sus archivos
correspondientes. Para que funcione correctamente:

ESPECTROFOTÓMETRO:
  El nombre base de la muestra debe coincidir entre:
    sample_<NOMBRE>-xxx.Sample.asc
    
FTIR:
  El nombre de la pestaña debe coincidir con el nombre de la muestra:
    sample_<NOMBRE>-xxx
    
El programa hace comparaciones flexibles, eliminando espacios, puntos, 
guiones y convirtiendo a minúsculas.

Ejemplo de coincidencia:
  - sample_Material-A-001.Sample.asc    (Espectrofotómetro)
  - sample_Material-A-001               (Pestaña FTIR)
  Estos archivos se procesarán juntos para la misma muestra.


4.3 FORMATO DE FECHAS
─────────────────────
Para archivos FTIR, las fechas deben estar en formato:
  YYYYMMDD
  
Ejemplo: 20260126data.xlsx (26 de enero de 2026)

5. ESTRUCTURA DE CARPETAS
═══════════════════════════════════════════════════════════════════════════════

Al iniciar el programa, se solicitará:

1. Ubicación y nombre del archivo Excel de salida
2. Carpeta con los archivos del Espectrofotómetro (si aplica)
   - Puede tener subcarpetas
   - El programa buscará recursivamente archivos .asc
3. Carpeta con los archivos FTIR (si aplica)
   - Puede tener subcarpetas
   - El programa buscará recursivamente archivos .xlsx/.xls


Estructura recomendada:
  
  Proyecto/
  ├── Espectrofotometro/
  │   ├── sample_Muestra1-001.Sample.asc
  │   ├── sample_Muestra2-002.Sample.asc
  │   ├── base_20260126.BaseLine.asc
  │   ├── zero_20260126.ZeroLine.asc
  │   └── ventana_cuarzo.Sample.asc
  │
  └── FTIR/
      ├── 20260126data.xlsx
      │   └── (pestañas: sample_Muestra1-001, sample_Muestra2-002)
      └── 20260126ref.xlsx
          └── (pestañas: zero_, base_, ventana_)

6. CONFIGURACIÓN PERSONALIZADA
═══════════════════════════════════════════════════════════════════════════════

6.1 ARCHIVO DE CONFIGURACIÓN (config.ini)
──────────────────────────────────────────
Ubicación: user_templates/Configuracion/config.ini

Este archivo permite personalizar:
  - Rutas a las plantillas de Excel
  - Celdas donde se escriben los datos en cada plantilla
  - Número máximo de medidas
  - Nombres de las hojas de las plantillas

Para modificar la configuración:
  1. Abrir config.ini con un editor de texto
  2. Modificar los valores según necesidad
  3. Guardar el archivo
  4. Reiniciar el programa


6.2 PLANTILLAS DE EXCEL
────────────────────────
Ubicación: user_templates/plantillas/

Las plantillas disponibles son:
  - abs_plantilla.xlsx          (Absortancia)
  - refl_plantilla.xls          (Reflectancia)
  - trans_csp_plantilla.xls     (Transmitancia CSP)
  - trans_pv_plantilla.xls      (Transmitancia PV)

IMPORTANTE: NO modificar estas plantillas a menos que sea necesario. Si se
modifican, asegurarse de actualizar también el archivo config.ini con las
nuevas ubicaciones de celdas.

NOTAS ADICIONALES
═══════════════════════════════════════════════════════════════════════════════

- El programa utiliza cuadros de diálogo para solicitar información
- Todas las selecciones se confirman antes de procesar
- Se recomienda cerrar todos los archivos Excel antes de ejecutar el programa
- Para medidas con ventana, asegurarse de marcar la opción correspondiente
- Los archivos de referencia (base/zero) se seleccionan automáticamente según
  la fecha más cercana a la medida de la muestra

═══════════════════════════════════════════════════════════════════════════════
SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════════

Si el programa no encuentra las muestras:
  ✓ Verificar que los nombres de archivos sigan las convenciones
  ✓ Verificar que los prefijos sean correctos (sample_, base_, zero_, etc.)
  ✓ Comprobar que las extensiones sean correctas (.asc, .xlsx, .xls)
  ✓ Revisar que las carpetas seleccionadas contengan los archivos

Si hay errores de Excel:
  ✓ Cerrar todos los archivos Excel abiertos
  ✓ Verificar que las plantillas existan en user_templates/plantillas/
  ✓ Comprobar que config.ini tenga las rutas correctas

Para más ayuda:
  ✓ Revisar el archivo de log de errores (.txt)
  ✓ Verificar que references.xlsx contenga todas las pestañas necesarias
